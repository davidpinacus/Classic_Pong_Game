# Classic Pong Game (Local Multiplayer)

A retro-style, local two-player Pong game built from scratch using Python's **Turtle graphics library**. Experience the classic arcade game with a modern twist—the ball speeds up with every successful volley!

## Features

* **Local Multiplayer:** Play head-to-head with a friend on the same keyboard.
* **Dynamic Difficulty:** The ball accelerates every time it hits a paddle, intensifying the gameplay.
* **Clean OOP Architecture:** Built using clean, modular Object-Oriented Programming (OOP) principles in Python.

---

##  How to Play

The objective is simple: deflect the ball past your opponent's paddle to score points. 

| Player | Move Up | Move Down |
| :--- | :---: | :---: |
| **Left Player** | <kbd>W</kbd> | <kbd>S</kbd> |
| **Right Player** | <kbd>▲</kbd> (Up Arrow) | <kbd>▼</kbd> (Down Arrow) |
