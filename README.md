A simple, local-multiplayer Pong game built using Python's Turtle graphics library.
## 🎮 How to Play
- **Left Player (W/S):** Control the left paddle.
- **Right Player (Up/Down):** Control the right paddle.
- The ball speeds up every time it hits a paddle!

## 📂 Files
- `game.py`: Main game loop and setup.
- `paddle.py`: Paddle logic and movement.
- `ball.py`: Ball physics and collisions.
- `scoreboard.py`: Score tracking and UI.
- `center_line.py`: Dashed court divider.

## 🚀 Setup
Run `python game.py` to start the game.
